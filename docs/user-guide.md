## 2. 租户管理

### 2.1 创建租户

**步骤：**

1. 进入「租户管理」→「租户列表」
2. 点击「新建租户」按钮
3. 填写租户信息：
   - **租户名称**：显示名称（如：Acme 公司）
   - **子域名**：唯一标识（如：acme，将生成 acme.yourdomain.com）
   - **自定义域名**：（可选）如 www.acme.com
   - **套餐**：选择服务套餐
   - **管理员邮箱**：租户管理员账号
4. 点击「创建」

**示例：**

```
租户名称：示例科技有限公司
子域名：example
自定义域名：www.example.com
套餐：专业版
管理员邮箱：admin@example.com
```

### 2.2 配置租户设置

**基础设置：**
- 网站标题
- Logo 上传
- Favicon 设置
- 联系信息

**SEO 设置：**
- 网站描述
- 关键词
- 社交媒体卡片

**功能开关：**
- 启用/禁用电商功能
- 启用/禁用客服功能
- 启用/禁用 AI 功能

### 2.3 租户状态管理

| 状态 | 说明 | 操作 |
|------|------|------|
| 激活 | 正常运行 | 可访问 |
| 未激活 | 待启用 | 不可访问 |
| 暂停 | 临时停用 | 显示维护页面 |
| 部署中 | 正在初始化 | 等待完成 |

### 2.4 自定义域名配置

1. 在租户设置中添加自定义域名
2. 在 DNS 服务商处添加 CNAME 记录：
   ```
   Type: CNAME
   Name: www
   Value: yourdomain.com
   TTL: 3600
   ```
3. 等待 DNS 生效（通常 5-30 分钟）
4. 点击「验证域名」
5. 配置 SSL 证书（自动或手动上传）

## 3. 主题配置

### 3.1 选择主题

1. 进入「主题管理」
2. 浏览可用主题：
   - 企业官网主题
   - 电商商城主题
   - 博客资讯主题
   - 作品展示主题
3. 点击「预览」查看效果
4. 点击「应用」启用主题

### 3.2 主题自定义

**颜色配置：**
```
主色调：#1890ff（品牌主色）
辅色调：#52c41a（成功/积极）
警告色：#faad14（提醒/警告）
错误色：#f5222d（错误/危险）
背景色：#f0f2f5（页面背景）
```

**字体配置：**
- 标题字体：思源黑体 / Roboto
- 正文字体：微软雅黑 / Open Sans
- 代码字体：Consolas / Monaco

**布局配置：**
- 页面宽度：全宽 / 固定宽度
- 导航位置：顶部 / 侧边
- 页脚样式：简洁 / 完整

### 3.3 上传自定义主题

1. 准备主题文件（ZIP 格式）
2. 进入「主题管理」→「上传主题」
3. 选择主题文件
4. 填写主题信息：
   - 主题名称
   - 版本号
   - 作者信息
   - 主题描述
5. 点击「上传并安装」

### 3.4 主题导出/导入

**导出主题：**
1. 选择要导出的主题
2. 点击「导出」
3. 下载主题文件（包含所有配置）

**导入主题：**
1. 点击「导入主题」
2. 选择之前导出的主题文件
3. 确认覆盖或创建新版本

## 4. 内容管理

### 4.1 内容区块管理

**创建内容区块：**

1. 进入「内容管理」→「内容区块」
2. 点击「新建区块」
3. 填写信息：
   - **区块标识**：唯一 key（如：home_hero）
   - **区块名称**：显示名称
   - **区块类型**：富文本 / Markdown / JSON
   - **内容**：具体内容
   - **语言**：zh-CN / en / 其他
4. 点击「保存」

**常用区块示例：**

```
home_hero - 首页 Hero 区域
home_features - 首页特性展示
home_about - 首页关于我们
footer_contact - 页脚联系信息
```

### 4.2 页面管理

**创建页面：**

1. 进入「内容管理」→「页面」
2. 点击「新建页面」
3. 配置页面：
   - **页面标题**：浏览器标题
   - **页面 URL**：自定义 slug（如：about-us）
   - **页面模板**：选择布局模板
   - **页面内容**：编辑器输入
   - **SEO 信息**：Meta 标题/描述/关键词
   - **发布状态**：草稿 / 已发布
4. 点击「保存」或「发布」

**页面模板：**

| 模板 | 用途 |
|------|------|
| 默认 | 通用页面 |
| 全宽 | 大图/视频展示 |
| 侧边栏 | 带侧边导航 |
|  landing | 营销落地页 |
| 空白 | 完全自定义 |

### 4.3 文章管理（CMS）

**发布文章：**

1. 进入「内容管理」→「文章」
2. 点击「写文章」
3. 填写内容：
   - 标题
   - 分类
   - 标签
   - 封面图
   - 摘要
   - 正文（富文本编辑器）
4. 设置发布选项：
   - 立即发布 / 定时发布
   - 允许评论
   - 置顶
5. 点击「发布」

**文章分类：**

```
新闻动态
├── 公司新闻
├── 行业资讯
└── 产品更新
帮助中心
├── 使用指南
├── 常见问题
└── 视频教程
```

### 4.4 媒体库

**上传文件：**

1. 进入「媒体库」
2. 点击「上传文件」或拖拽文件
3. 支持格式：
   - 图片：JPG, PNG, GIF, WebP, SVG
   - 文档：PDF, DOC, XLS
   - 视频：MP4, WebM
   - 音频：MP3, WAV
4. 添加文件信息：
   - 标题
   - 描述
   - 替代文本（SEO）

**批量操作：**
- 批量删除
- 批量移动
- 批量下载

## 5. AI 功能使用

### 5.1 AI 内容生成

**生成文章：**

1. 进入「AI 助手」→「内容生成」
2. 选择生成类型：
   - 博客文章
   - 产品描述
   - 营销文案
   - 社交媒体帖子
3. 输入提示词（Prompt）：
   ```
   主题：人工智能在医疗领域的应用
   字数：1000字
   风格：专业、易懂
   关键词：AI医疗、诊断、治疗
   ```
4. 点击「生成」
5. 预览并编辑生成的内容
6. 点击「保存到草稿」或「直接发布」

**生成产品描述：**

1. 在商品编辑页面
2. 点击「AI 生成描述」
3. 输入产品关键信息：
   - 产品名称
   - 主要特点
   - 目标受众
4. 选择描述风格：
   - 简洁型（50字）
   - 标准型（100字）
   - 详细型（200字）
5. 点击「生成」并应用

### 5.2 AI 翻译

**翻译内容：**

1. 选择要翻译的内容
2. 点击「AI 翻译」
3. 选择目标语言：
   - 英语
   - 日语
   - 韩语
   - 法语
   - 德语
   - 西班牙语
4. 选择翻译风格：
   - 正式
   - 商务
   - 轻松
5. 预览翻译结果
6. 确认保存

**批量翻译：**
- 支持批量翻译产品信息
- 支持批量翻译页面内容
- 支持批量翻译分类名称

### 5.3 AI SEO 优化

**优化建议：**

1. 进入「AI 助手」→「SEO 优化」
2. 选择要优化的页面
3. 点击「分析」
4. 查看优化建议：
   - 标题优化
   - 描述优化
   - 关键词建议
   - 内容改进
5. 一键应用建议

### 5.4 AI 客服配置

**训练知识库：**

1. 进入「客服系统」→「知识库」
2. 添加常见问题：
   - 问题
   - 答案
   - 相关标签
3. 上传文档（PDF、Word）
4. 点击「训练 AI」
5. 测试机器人回答

**配置自动回复：**

1. 设置触发条件：
   - 关键词匹配
   - 意图识别
   - 时间段
2. 配置回复内容：
   - 直接回答
   - 转人工
   - 创建工单
3. 设置转人工阈值：
   - 无法回答时
   - 用户要求时
   - 复杂问题时

## 6. 电商功能

### 6.1 商品管理

**添加商品：**

1. 进入「电商管理」→「商品」
2. 点击「添加商品」
3. 填写基本信息：
   - 商品名称
   - 商品分类
   - 商品标签
   - 商品图片（主图+详情图）
4. 设置价格和库存：
   - 售价
   - 原价（用于显示折扣）
   - 成本价（内部使用）
   - SKU 编码
   - 库存数量
5. 填写商品详情：
   - 商品描述
   - 规格参数
   - 使用说明
6. 设置 SEO 信息
7. 点击「保存」或「上架」

**商品分类：**

```

├── 电子产品
│   ├── 手机
│   ├── 电脑

│   └── 鞋靴
```

### 6.2 订单管理

**订单流程：**

```
待付款 → 待发货 → 已发货 → 已完成
   ↓        ↓         ↓
 已取消   已退款    退货中
```

**处理订单：**

1. 进入「电商管理」→「订单」
2. 查看订单列表
3. 点击订单查看详情
4. 执行操作：
   - 确认订单
   - 发货（填写物流信息）
   - 退款
   - 备注

**批量操作：**
- 批量发货（导出 Excel）
- 批量打印快递单
- 批量导出订单

### 6.3 促销活动

**创建促销：**

1. 进入「电商管理」→「促销」
2. 选择促销类型：
   - 限时折扣
   - 满减活动
   - 优惠券
   - 包邮活动
3. 配置规则：
   - 活动时间
   - 参与商品
   - 优惠力度
   - 使用限制
4. 点击「创建活动」

## 7. 客服系统

### 7.1 配置客服渠道

**网页组件：**

1. 进入「客服系统」→「渠道管理」
2. 点击「创建渠道」
3. 选择「网页组件」
4. 配置外观：
   - 主题颜色
   - 欢迎语
   - 自动回复
5. 获取嵌入代码
6. 将代码添加到网站 HTML

**配置示例：**

```html
<!-- 添加到网站 <head> -->
<script src="https://yourdomain.com/chat-widget.js"></script>
<script>
  BuildAIChat.init({
    channelId: 'your-channel-id',
    theme: 'light',
    position: 'right-bottom'
  });
</script>
```

### 7.2 客服工作台

**接待访客：**

1. 登录客服工作台
2. 设置在线状态：在线 / 忙碌 / 离线
3. 查看访客列表
4. 点击访客开始对话
5. 使用快捷回复
6. 标记会话状态

**快捷回复：**

```
#欢迎 - 欢迎语
#运费 - 运费说明
#退换 - 退换货政策
#工作时间 - 客服工作时间
```

### 7.3 工单系统

**创建工单：**

1. 访客提交问题
2. 系统自动分类
3. 分配给客服
4. 客服处理并回复
5. 访客确认解决
6. 工单关闭

**工单状态：**

| 状态 | 说明 |
|------|------|
| 待处理 | 新创建的工单 |
| 处理中 | 客服正在处理 |
| 待回复 | 等待访客回复 |
| 已解决 | 问题已解决 |
| 已关闭 | 工单结束 |

## 8. 常见问题

### 8.1 登录问题

**Q: 忘记密码怎么办？**

A: 
1. 点击登录页面的「忘记密码」
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件链接设置新密码

**Q: 收不到验证码？**

A:
1. 检查垃圾邮件文件夹
2. 确认邮箱地址正确
3. 等待 60 秒后重试
4. 联系管理员检查邮件服务

### 8.2 网站访问问题

**Q: 网站显示 404？**

A:
1. 检查页面是否已发布
2. 检查 URL 是否正确
3. 检查路由配置
4. 清除 CDN 缓存

**Q: 网站加载慢？**

A:
1. 检查图片大小，压缩后重新上传
2. 启用 CDN 加速
3. 检查网络连接
4. 联系技术支持

### 8.3 功能使用问题

**Q: AI 生成功能不可用？**

A:
1. 检查 AI API Key 是否配置
2. 检查 API Key 余额
3. 检查网络连接
4. 查看系统日志

**Q: 邮件发送失败？**

A:
1. 检查 SMTP 配置
2. 检查邮箱密码/授权码
3. 检查邮箱服务商设置
4. 查看发送日志

### 8.4 数据问题

**Q: 如何备份数据？**

A:
1. 进入「系统设置」→「备份恢复」
2. 点击「立即备份」
3. 或设置自动备份计划

**Q: 误删数据如何恢复？**

A:
1. 进入「系统设置」→「备份恢复」
2. 选择备份文件
3. 点击「恢复」
4. 注意：恢复会覆盖当前数据

### 8.5 集成问题

**Q: 如何接入第三方系统？**

A:
1. 进入「系统设置」→「API 管理」
2. 生成 API Key
3. 查看 API 文档
4. 按照文档进行集成

**Q: 支持哪些支付方式？**

A:
- 微信支付
- 支付宝
- PayPal
- Stripe
- 银行转账（手动确认）
