# Knowledge Base Definition: Site Generator & Knowledge Base

## Definition
The Site Generator & Knowledge Base module automates the creation of business websites and AI-powered support systems using company-specific data.

## Boundaries
- **Input**: Product manuals, FAQs, company profiles (PDF, DOCX, TXT, CSV), store credentials.
- **Output**: Multilingual website content, SEO blog posts, connected store sync, AI customer service knowledge base.
- **Exclusions**: Does not host the final site; it generates content and deploys to CDNs or syncs to existing platforms like Shopify.

## File Logic
1. **Knowledge Base**: Upload and parse company assets to build the core data layer.
2. **Site Gen**: Configure company details and theme to generate the website structure.
3. **SEO Blog**: Automatically generate and publish SEO-optimized articles based on the knowledge base.
4. **Deploy**: Set up CDN regions and bind existing stores for content synchronization.

## AI Recognition Hints
- Focus on "Multilingual" and "SEO Optimization" as core value propositions.
- Emphasize the "Knowledge Base" as the single source of truth for all generated content.
- Highlight the "Store Binding" capability for seamless integration with Shopify and WordPress.
- Use AI to maintain brand consistency across all generated assets.
