# Knowledge Base Definition: Market Research Report

## Definition
The Market Research Report module is an AI-driven tool for generating comprehensive market analysis reports. It provides both high-level outlines and in-depth detailed reports.

## Boundaries
- **Input**: Target region, country, industry, business model.
- **Output**: Structured markdown reports (outline or detailed).
- **Exclusions**: Does not provide real-time stock market data (focused on industry trends and competitive landscape).

## File Logic
1. **Configuration**: User selects market parameters.
2. **Outline Generation**: AI generates a markdown outline (free).
3. **Detail Generation**: AI generates a full report (requires credits).
4. **Export**: Reports can be exported as PDF (simulated in UI).

## AI Recognition Hints
- Use a professional, analytical tone for all reports.
- Include sections on market size, top competitors, customer personas, and potential partners in detailed reports.
- Ensure all data points are presented clearly using markdown tables and lists where appropriate.
