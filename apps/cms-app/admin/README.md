# CMS App Admin

基于 React + Ant Design 的内容管理后台。

## 功能模块

- 📊 **仪表盘** - 数据统计、最近文章
- 📝 **文章管理** - 文章CRUD、AI辅助写作
- 📂 **分类管理** - 层级分类管理
- 🏷️ **标签管理** - 标签CRUD
- 🖼️ **媒体库** - 文件上传、图片管理
- 🤖 **AI助手** - AI写文章、生成摘要、SEO优化
- ⚙️ **系统设置** - 站点设置、SEO设置、AI设置

## 快速开始

```bash
# 安装依赖
npm install

# 开发服务器
npm run dev

# 构建
npm run build
```

## 环境变量

创建 `.env` 文件:

```env
VITE_API_URL=http://localhost:3000/api/v1
```

## 访问地址

- 后台地址: http://localhost:5174 (默认)
- API文档: http://localhost:3000/api/docs
